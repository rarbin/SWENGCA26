Using XTiles for this example.\\

Good things:
\begin{itemize}
  \item You can swap between different accounts
  \item Home page, inbox, calendar, search layout
  \item Syncs across devices
  \item Organises calendars, tasks, notes under one planner
\end{itemize}

Bad things:
\begin{itemize}
  \item A lot of features are behind an expensive paywall
  \item Some features don't work like sharing a project
\end{itemize}

Product Vision Layout for XTiles:
\begin{enumerate}
  \item FOR (target customer)
  \item WHO (statement of the need or opportunity)
  \item The (PRODUCT NAME) is a (product category)
  \item THAT (key benefit)
  \item UNLIKE (primary competitive alternative)
  \item OUR PRODUCT (primary differentiation)
\end{enumerate}
