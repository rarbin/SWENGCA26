# SWENGCA26

Using the github client, clone this repo with 'gh repo clone rarbin/SWENGCA26'.
This will put you on the master branch by default.

Use git checkout -b BRANCHNAME to move to the relevant branch, e.g. Similar App A1.

Add to repos with git add FILENAME or git add . to add everything in the current directory.
git push can then be used to push to selected branch

Once a branch has been merged with master and is no longer needed, 
delete a branch with git branch -d BRANCHNAME.
